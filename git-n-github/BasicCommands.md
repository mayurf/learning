# Brief rundown of basic commands and convenient commands for common tasks

Installing git is considered trivial and is left as an exercise to the reader :D

1. Setting user name
   ```console
   name@localhost:foo$ git config --global user.name "Some Dude"
   ```
2. Setting user email
   ```console
   name@localhost:foo$ git config --global user.email "somedude@mail.com"
   ```
3. Color code git console output
   ```console
   name@localhost:foo$ git config --global color.ui auto
   ```
